# Bài 1 - Quản lý tệp và thư mục

## 1. Mục tiêu bài tập

Bài tập này giúp tôi rèn luyện kỹ năng thao tác cơ bản với tệp tin và thư mục, biết cách tổ chức dữ liệu học tập một cách khoa học, dễ tìm kiếm và dễ quản lý.

Thông qua bài tập, tôi học cách xây dựng cấu trúc thư mục hợp lý, đặt tên file nhất quán và lưu trữ tài liệu sao cho thuận tiện cho quá trình học tập sau này.

## 2. Quá trình thực hiện

- Xác định các nhóm tài liệu cần quản lý.
- Tạo cấu trúc thư mục theo từng chủ đề hoặc từng bài học.
- Sắp xếp file vào đúng thư mục tương ứng.
- Đặt tên file theo quy tắc thống nhất, dễ hiểu.
- Chụp ảnh minh chứng cấu trúc thư mục và quy tắc đặt tên.
- Tổng hợp kết quả vào file PDF sản phẩm cuối cùng.

## 3. Sản phẩm cuối cùng

Sản phẩm cuối cùng của bài tập là file PDF trình bày cấu trúc thư mục, quy tắc đặt tên file và ảnh minh chứng quá trình tổ chức dữ liệu.

File sản phẩm:
- `san_pham/Bài_1.pdf`

Trạng thái kiểm tra:
- PDF hiện có trong thư mục `san_pham/`.
- Portfolio mới không sử dụng ảnh/PDF minh chứng từ bài này trong giao diện hiện tại.

## 4. Minh chứng có trong PDF

PDF sản phẩm đã chứa các minh chứng như:

- Ảnh chụp cấu trúc thư mục đã tạo.
- Ảnh minh họa cách đặt tên file.
- Mô tả quy tắc tổ chức thư mục.
- Giải thích lý do lựa chọn cách sắp xếp dữ liệu.

Chưa chọn trang minh chứng cho portfolio mới.

## 5. Kỹ năng đã học được

- Biết cách tổ chức tệp và thư mục khoa học.
- Biết đặt tên file rõ ràng, nhất quán.
- Biết lưu trữ tài liệu học tập theo cấu trúc dễ tìm kiếm.
- Hiểu tầm quan trọng của quản lý dữ liệu cá nhân.
- Biết trình bày quá trình thực hiện bằng minh chứng trực quan.

## 6. Điểm muốn nhấn mạnh trên website

Trên website, cần nhấn mạnh rằng bài tập này không chỉ là tạo thư mục, mà là bước đầu hình thành thói quen quản lý dữ liệu học tập một cách chuyên nghiệp.

Nếu cần đưa bài này vào website sau này, ưu tiên hiển thị trang có:
- cấu trúc thư mục,
- quy tắc đặt tên file,
- phần giải thích lý do tổ chức dữ liệu.
